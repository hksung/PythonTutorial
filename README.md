- Designed for LING 351 (Language Technology and LLMs) and LING 581 (Natural Language Processing I) at RIT, Fall 2025.
- Updated for SLS 775 (NLP in corpus research) at UH, Fall 2026.

### Local preview with Jekyll

1. Install the site dependencies:

	```bash
	bundle config set --local path 'vendor/bundle'
	bundle install
	```

2. Start the local Jekyll server:

	```bash
	bundle exec jekyll serve
	```

3. Open the site in your browser:

	```text
	http://127.0.0.1:4000/PythonTutorial/
	```

4. To open a specific page such as module 2:

	```text
	http://127.0.0.1:4000/PythonTutorial/2.html
	```
