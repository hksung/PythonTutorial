# Tuples

A tuple is like a list but immutable.

```python
t = (1, 2)
t = t + (3, 4)
print(t)

print(t[0])
print(t[-1])
print(t[1:3])

x, y, *rest = t
print(x, y, rest)
```

Use tuples for fixed values or for returning multiple results.
