# Methods

Methods are functions attached to an object.

```python
sample = "This is a STRING"
```

## `.lower()`

```python
sample.lower()
```

## `.split()`

```python
sample.split()
sample.lower().split()
```

## Combining functions and methods

```python
len(sample.lower().split())
```

## Practice

```python
my_text = "Python is Fun"

print(my_text.lower())
print(my_text.split())
print(len(my_text.split()))
```
