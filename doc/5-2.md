# Lemmatization

Lemmatization converts each token to its base dictionary form.

## Dictionary-based lemmatizer

```python
def lemmatize_dict(tokens, lemma_dict):
    return [lemma_dict.get(tok, tok) for tok in tokens]

lemma_dict = {"is": "be", "running": "run"}
print(lemmatize_dict(["this", "is", "running"], lemma_dict))
```

## NLTK WordNetLemmatizer

```python
from nltk.stem import WordNetLemmatizer

wnl = WordNetLemmatizer()
print(wnl.lemmatize("running", pos="v"))
print(wnl.lemmatize("better", pos="a"))
```

## spaCy lemmatizer

```python
import spacy
nlp = spacy.load("en_core_web_sm")

def lemmatize_spacy(text):
    doc = nlp(text)
    return [token.lemma_ for token in doc]

print(lemmatize_spacy("Running faster makes you better."))
```
