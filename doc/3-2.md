# Lists

A list is an ordered, mutable collection.

```python
platforms = ["Windows", "Mac", "Linux"]
```

## Indexing and slicing

```python
print(platforms[1])
print(platforms[1:])
```

## Concatenation

```python
numbers = [1, 2, 3]
combined = platforms + numbers
print(combined)
```

## Mutability

```python
platforms.append("Android")
print(platforms)

platforms[0] = "FreeBSD"
print(platforms)
```
