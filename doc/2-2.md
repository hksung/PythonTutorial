# Variables

Variables store values under names.

```python
language = "Python"
year = 2026
score = 98.5
```

You can reuse a variable name later.

```python
message = "I am learning"
message = message + " Python"
print(message)
```

## Practice

```python
favorite_food = "pasta"
times_per_week = 2

print(favorite_food)
print(times_per_week)
```
