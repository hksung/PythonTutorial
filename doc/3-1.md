# Start with strings

You can treat a string like a sequence of characters.

## Indexing

```python
sample = "sample"

print(sample[0])
print(sample[1])
print(sample[-1])
```

## Slicing

```python
s = "sample"

print(s[0:1])
print(s[0:2])
print(s[1:-1])
print(s[1:])
print(s[:2])
```

## Concatenation and membership

```python
a = "super"
b = "awesome"

print(a + b)
print(a + " " + b)
```

```python
word = "awesome"

if "a" in word:
    print("Contains 'a'!")
```
