# Frequency calculation

Text analysis often starts with counting how many times each token or lemma appears.

## Using `collections.Counter`

```python
from collections import Counter

tokens = ["a", "b", "a", ".", "a", "b"]
freq = Counter(tokens)
print(freq)
print(freq.most_common(3))
```

## Simple loop version

```python
def freq_simple(tokens):
    counts = {}
    for tok in tokens:
        counts[tok] = counts.get(tok, 0) + 1
    return counts

print(freq_simple(tokens))
```

## End-to-end example

```python
from collections import Counter
import spacy

nlp = spacy.load("en_core_web_sm")

def tokenize_and_lemmatize(text):
    doc = nlp(text)
    return [token.lemma_.lower() for token in doc if not token.is_space and not token.is_punct]

def corpus_freq(filepath):
    total = Counter()
    with open(filepath, encoding="utf-8", errors="ignore") as f:
        text = f.read()
    lems = tokenize_and_lemmatize(text)
    total.update(lems)
    return total
```
