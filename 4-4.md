# Classes

Classes group data and behavior together.

```python
class Animal(object):
    def __init__(self, species, age):
        self.species = species
        self.age = age

    def is_person(self):
        return self.species

    def age_one_year(self):
        self.age += 1
```

Inheritance lets you build a new class from an existing one.

```python
class Dog(Animal):
    def age_one_year(self):
        self.age += 7
```

```python
a = Animal("Human", 5)
d = Dog("Dog", 5)

a.age_one_year()
d.age_one_year()

print(a.age)
print(d.age)
```
