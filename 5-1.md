# Tokenization

Tokenization is the process of converting a raw text string into smaller units called tokens.

## Simple `.split()`

```python
text = "Hello, world! This is Python."
tokens = text.split(" ")
print(tokens)
```

This is simple, but punctuation stays attached to words.

## A simple custom tokenizer

```python
def tokenize_simple(text):
    for p in [".", ",", "!", "?", ";", ":"]:
        text = text.replace(p, f" {p} ")
    return text.lower().split()

print(tokenize_simple(text))
```

## Regex-based tokenization

```python
import re

def tokenize_regex(text):
    return re.findall(r"\w+|[^\w\s]", text.lower())

print(tokenize_regex(text))
```

## Toolkit-based tokenization

### NLTK

```python
from nltk.tokenize import word_tokenize

def tokenize_nltk(text):
    return word_tokenize(text.lower())
```

### spaCy

```python
import spacy
nlp = spacy.load("en_core_web_sm")

def tokenize_spacy(text):
    doc = nlp(text)
    return [token.text for token in doc]
```
