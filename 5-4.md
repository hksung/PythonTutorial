# Exercises

Use the local practice file [corpus/freq_test.txt](/Users/hakyungsung/Documents/03_Repository/01_GitHub/PythonTutorial/corpus/freq_test.txt) for these exercises.

## Exercise 1. Read the local text file

```python
with open("corpus/freq_test.txt", "r", encoding="utf-8") as f:
	raw = f.read()

print(type(raw))
print(len(raw))
print(raw[:120])
```

Question:

- What do you notice in the first 120 characters?

## Exercise 2. Tokenize the text

Choose one tokenizer from this module and apply it to `raw`.

Starter example:

```python
tokens = tokenize_simple(raw)
print(tokens[:30])
print(len(tokens))
```

## Exercise 3. Reuse the token list

Store the tokenized output in a variable so you can reuse it in the next steps.

## Exercise 4. Lemmatize the text

Lemmatize the text and store the result in a new variable.

If you use spaCy, your code can look like this:

```python
lemmas = lemmatize_spacy(raw)
print(lemmas[:30])
```

## Exercise 5. Calculate word or lemma frequencies

Count the frequency of the tokens or lemmas.

Starter example:

```python
freq = freq_simple(lemmas)
print(freq)
```

You can also use `Counter` if you prefer.

## Exercise 6. Inspect the results

Print:

- the 20 most common lemmas
- the 20 least frequent lemmas

Starter example with `Counter`:

```python
from collections import Counter

lemma_freq = Counter(lemmas)
print(lemma_freq.most_common(20))
print(lemma_freq.most_common()[-20:])
```

## Exercise 7. Reflect

Write 2 or 3 observations about the most frequent and least frequent lemmas.

Helpful questions:

- Which words appear very often?
- Are punctuation marks or function words common?
- Do you notice any spelling or grammar issues in the original text that affect the frequency list?
